# API-INTEGRATION

A simple and responsive Weather App built using HTML, CSS, and JavaScript. This project uses the OpenWeatherMap API to fetch real-time weather data such as temperature, humidity, and weather conditions based on the city entered by the user.

Features:
• Real-time weather information
• API Integration using Fetch API
• Responsive user interface
• Error handling for invalid city names
• Displays temperature, humidity, and weather condition

Technologies Used:
• HTML
• CSS
• JavaScript
• OpenWeatherMap API
